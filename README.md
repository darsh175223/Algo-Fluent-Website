# Algofluent

Algofluent is a dynamic website designed to visualize various data structures and algorithms. It features interactive visualizations for data structures like arrays, hashmaps, and binary search trees, as well as algorithm visualizations for selection sort, linear regression, logistic regression. The project also includes user authentication using the MERN stack and programming tutorials.

## Features

### Data Structure Visualizations
- **Linked Lists**: Add or delete elements and see the structure update in real-time.
- **BST**: Add or delete elements and see the structure update in real-time.
- **Closed Hashes**: Generate hashmaps of any size and visualize the placement and retrieval of elements.
- **Open Hashes**: Visualize the placement and retrieval of elements.

### Algorithm Visualizations
- **Selection Sort**: An interactive visualizer with a slider to adjust the array size, a sort button, and a bar graph to display the array.
- **Quick Sort**: An interactive visualizer with a slider to adjust the array size, a sort button, and a bar graph to display the array.
- **Merge Sort**: An interactive visualizer with a slider to adjust the array size, a sort button, and a bar graph to display the array.
- **Bubble Sort**: An interactive visualizer with a slider to adjust the array size, a sort button, and a bar graph to display the array.
- **Radix Sort**: An interactive visualizer with a slider to adjust the array size, a sort button, and a bar graph to display the array.

### Machine Learning Visualizations

- **Linear Regression**: Add points to an interactive coordinate graph, which generates and updates a regression line automatically.
- **Logistic Regression**: Visualize logistic regression using Chart.js and the `ml-logistic-regression` library.
- **Handwritten Digit Recognition**: Draw a number on a canvas and have it recognized by a Python-based program.
- **PongAI**: Play pong with AI.


### Additional Features
- **User Authentication**: MERN stack implementation for user sign-in and account creation.
- **Programming Tutorials**: Learn programming languages from Python to HTML -- available to registered users.

## Project Structure

- **Frontend**: Built with ReactJS.
  - **Data Structure and Algorithm Visualizers**: Various components for visualizing data structures and algorithms.
  - **User Authentication**: Components for sign-in and account creation pages.
- **Backend**: Node.js and Express.
  - **User Authentication**: RESTful APIs for managing user data.
- **Deployment**: Deployed using Docker, Render, and Netlify.
